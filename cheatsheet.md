# Neovim Cheatsheet

## ファイルツリー (nvim-tree)

| キー | 動作 |
|------|------|
| `-` | ファイルツリーをトグル |
| `q` | ファイルツリーを閉じる |
| `Ctrl+w h` | ファイルツリーへ移動 |
| `Ctrl+w l` | エディタへ移動 |

## コード折り畳み (nvim-ufo)

| キー | 動作 |
|------|------|
| `za` | カーソル位置の fold をトグル |
| `zR` | すべて展開 |
| `zM` | すべて折り畳む |
| `zr` | 1段階展開 |
| `zm` | 1段階折り畳む |
